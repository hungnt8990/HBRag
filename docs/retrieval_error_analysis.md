# Retrieval Error Analysis

Status: template only.

## Error Classes To Track

- wrong document
- wrong row
- wrong article
- missing header/title context
- placeholder table leakage
- footer/signature leakage
- artifact not selected before chunk fallback

## What To Capture

- query text
- requested profile
- selected profile
- top-k results
- source flags
- rerank scores
- artifact hits
- fallback reason
